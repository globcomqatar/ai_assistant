# AI Assistant for ERPNext

Natural language AI chat inside ERPNext Desk. Accepts English/Urdu, converts to ERP actions.

## Features
- ChatGPT-style UI inside ERPNext Desk
- Creates customers, quotations, invoices, job cards via natural language
- OpenAI integration with configurable model and API key
- Per-user monthly budget control and usage logging
- Floating launcher button on every desk page
